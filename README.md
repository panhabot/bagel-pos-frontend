<!-- GETTING STARTED -->
## Getting Started


### Installation


1. Clone the repo
   ```sh
   git clone https://github.com/panhabot/bagel-pos-frontend.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Run Project
   ```sh
   npm run dev
   ```
